# Project Name: Where is my key?

### Website: https://storage.zack-andy.site/

### Website Introduction: 提供使用者自由紀錄/規劃房間及物品的介面，並以列表展現最終結果

### Functions:  提供使用者編輯介面，能夠在空間內放入家具、物品，並以列表展示結果，讓使用者日後能較快速地找到物品的所在之處

---
